# Instrucciones para agregar tu CV PDF

Para agregar tu CV en formato PDF, tienes dos opciones:

## Opción 1: Usar el script automático (Recomendado)

### En Git Bash:
```bash
bash copiar-cv.sh
```

### En PowerShell:
```powershell
.\copiar-cv.ps1
```

El script buscará automáticamente el archivo "CV Juan Felipe Henao Tovar.pdf" y lo copiará a esta carpeta.

## Opción 2: Copia manual

1. Localiza tu archivo "CV Juan Felipe Henao Tovar.pdf"
2. Cópialo a esta carpeta (`/public/`)
3. Renómbralo como `cv.pdf`
4. El botón "Descargar CV" en la página web funcionará automáticamente

**Nota:** Si el archivo no existe, el botón intentará descargar pero no funcionará. Asegúrate de tener el PDF en esta ubicación.
